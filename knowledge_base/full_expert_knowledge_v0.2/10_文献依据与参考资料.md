## 5. RAG 与外部知识库

对应模块：

```text
README_使用说明.md
00_奖励函数设计Agent工作流.md
knowledge_index.json
```

RAG 的思想是把模型参数知识与外部可检索知识库结合。本系统中，专家知识库作为外部非参数知识，用于让 LLM 按任务类型调用奖励设计知识。

```text
Lewis, P., Perez, E., Piktus, A., et al. (2020).
```

## 11. 方法论中文表述建议

```text
本文构建了一个专家知识引导的奖励函数设计知识库，将奖励函数专家经验拆解为奖励骨架、任务到奖励的路由规则、奖励失败模式和奖励黑客检查清单。LLM Agent 在设计奖励函数时并非直接根据环境源码生成 reward，而是按照环境理解、知识调用、奖励方案设计、代码生成、训练诊断、奖励修改和经验记忆写入的流程进行迭代。
```

## 12. 方法论英文表述建议

```text
We construct an expert-knowledge-guided reward design knowledge base that decomposes reward design expertise into reward skeletons, task-to-reward routing rules, reward failure modes, and reward hacking checklists. The knowledge base is used by an LLM-based reward design agent through a structured workflow: environment understanding, reward skeleton retrieval, reward design planning, reward code generation, training feedback diagnosis, reward revision, and memory update.
```
